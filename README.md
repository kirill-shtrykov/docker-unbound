# Unbound Alpine Docker Image

This is unofficial Alpine image of [Unbound](https://nlnetlabs.nl/projects/unbound/about/) - validating, recursive, 
caching DNS resolver.

## Usage
```bash
docker run kirillshtrykov/unbound
```
